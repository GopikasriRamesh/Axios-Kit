Navigation Bar

The navigation bar is a primary layout component that provides site navigation and branding at the top of a page.

<nav class="navbar">
  <a href="#" class="navbar__brand">Axiom Kit</a>
  <ul class="navbar__links">
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
  <button class="btn btn--primary">Get Started</button>
</nav>
<!-- Add the .navbar--dark modifier class -->
<nav class="navbar navbar--dark">
  <!-- ... same content ... -->
</nav>
