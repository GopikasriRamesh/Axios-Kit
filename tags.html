Badges & Tags

Badges and tags are used to highlight an item's status for quick recognition. Use them to label, categorize, or count items.

<span class="badge">Default</span>
<span class="badge badge--success">Success</span>
<span class="badge badge--pill">Default</span>
<span class="badge badge--pill badge--error">Error</span>
<span class="badge badge--warning">Pending</span>
<span class="badge badge--info">In Progress</span>
.badge {
  display: inline-block;
  padding: 0.3em 0.6em;
  font-size: 0.8rem;
  font-weight: 700;
  line-height: 1;
  text-align: center;
  white-space: nowrap;
  vertical-align: baseline;
  border-radius: var(--border-radius);
  background-color: var(--subtle-bg);
  color: var(--secondary-text);
}
.badge--pill {
  border-radius: 50px; /* Creates the pill shape */
}
/* Uses notification color variables */
.badge--success {
  background-color: var(--color-success-bg);
  color: var(--color-success-text);
}
.badge--error {
  background-color: var(--color-error-bg);
  color: var(--color-error-text);
}
